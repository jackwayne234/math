# Math Study Materials

Self-study materials for algebra through trigonometry, with interactive HTML tools and practice problems.

## Contents

- `calculator.html` - Interactive calculator
- `graph_lines.html` / `graph_lines.py` - Line graphing tools
- `law_of_sines.html` - Law of sines interactive tool
- `practice_problems_algebra_through_trig.txt` - Practice problem sets
- `practice_problems_round2.txt` - Additional practice problems
- `PROGRESS.md` - Study progress tracker

## Usage

Open any `.html` file in a web browser to use the interactive tools.
